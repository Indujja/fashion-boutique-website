👗 Fashion Boutique Website
A simple web application for a fashion boutique to manage and display dress collections. Built with a user-friendly interface and powered by MongoDB for storing dress data and images.

✨ Features

🛍️ View a collection of dresses with images and descriptions

🧵 Add, edit, and delete dress details (admin access)

📦 Dresses stored in MongoDB, including image URLs or binary image data


🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js

Database: MongoDB

Image Handling: Stored as URLs or base64/binary in MongoDB GridFS


🚀 Getting Started

Prerequisites

Node.js installed

MongoDB running locally or via cloud (MongoDB Atlas)



Admins can:

Add new dresses with image uploads

Edit existing dress info

Delete outdated inventory
